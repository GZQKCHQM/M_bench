## 🔍 What is M_bench?

M_bench is a simple tool for testing how well your Apple Silicon Mac performs. It shows how fast your computer's CPU runs, how quickly memory moves data, and how well it handles multiple tasks at once. This helps you understand your Mac's strength using real data from tasks like running Python programs.

You do not need any technical skills to use M_bench. It works quietly in the background and gives clear results.

## 🖥️ System Requirements

Before you start, check the list below to be sure your Mac can run M_bench smoothly:

- Mac with Apple Silicon chip (M1, M2, or later)
- macOS version 11.0 (Big Sur) or higher
- At least 4 GB of free disk space

- Python 3.8 or later (required for some tuning features)

If you are not sure about your Mac’s specs:

## 📊 Understanding Your Results

M_bench breaks down the performance into three main parts:

- **CPU speed:** Measures your Mac’s raw processing power.
- **Memory bandwidth:** Shows how fast information moves between RAM and the CPU.
- **Parallelism:** Tests how well your Mac handles multiple tasks working at the same time.

You will see simple scores and charts. Higher scores mean better performance.

Use these results if you want to:

- Check how well your Mac runs heavy apps.
- Compare your Mac to others.
- Decide if you need to upgrade software or hardware.
- Tune Python programs for improved speed.

## ⚙️ How It Works

M_bench runs tests called *benchmarks*. These simulate real tasks your computer might do. Benchmarks measure the response time and speed of your Mac’s CPU and memory.

The app uses simple tests based on numbers and calculations. It also tests how your Mac runs many Python tasks at once. This helps if you work with data or code in Python, but you do not need to know Python to use M_bench.
